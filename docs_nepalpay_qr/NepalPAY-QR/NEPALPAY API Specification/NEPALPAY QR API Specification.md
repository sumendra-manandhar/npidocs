# 5. Background
This section includes technical specification of the interfaces for using NEPALPAY QR by the members
as Issuer or Acquirer for transction within the network or by technical members as Other Network for
interoperability of QR scheme/ network. This API Specification section illustrates the process flow for
QR code based payments in NEPALPAY QR including transactions and QR interoperability. It mainly
covers the followings:

1. High level process flow of NEPALPAY QR
2. Description of various sub-flows underlying the QR payments processing
3. API Specifications for members
