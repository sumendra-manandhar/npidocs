 # 1.	About this Document
This document intends to provide the clearing file details pertaining to card transactions that traverse the NPS National Card Switch (NPS - NCS). This document seeks to provide participating members with a deeper understanding of the operational dynamics governing card transaction clearing processes facilitated by the NPS - NCS platform. 

**1.1	Audience**
This manual is intended for technical staff and managers and customer support personnel of NCHL member banks and member PSOs.

**1.2	Organization of the Document**
Chapter 2, Introduction – This chapter provides overview of NPS-NCS.
Chapter 3, NPS-NCS Clearing File Specification – This chapter provides NCHL Clearing File Specification along with the list if fields.
Chapter 4, File Header Field Description – This chapter provides detailed description of each field of Clearing File Header.
Chapter 5, File Trailer Field Description – This chapter provides detailed description of each field of Clearing File Trailer.
Chapter 6, Transaction Field Description – This chapter provides detailed description of each field of Clearing File Transaction.
Annexure – This contains various supplement or appendix related with this document.
