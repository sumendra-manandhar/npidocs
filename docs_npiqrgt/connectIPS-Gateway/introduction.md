# 1. Introduction
## 1.1 Purpose
The purpose of the document is to describe necessary information for integrating merchant’s e-commerce 
site/mobile application to connectIPS Core Module to initiate e-payment transactions by its customer. 

## 1.2. Intended Audience and Reading Suggestions
The intended audience for this document comprises mainly software developers, software development 
managers or any other person with technical expertise in software development who is a Merchants’ technical 
partner or employee or has the Merchants’ approval for reading this document. And this process details are 
intended for the operations/ business managers of the merchants having e-commerce application.
