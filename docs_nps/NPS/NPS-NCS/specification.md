# 1. 	About this Document
 
 This document intends to provide the technical details of NCHL - National Payment Switch - National Card Switch (NPS - NCS) transaction processing provisions. This document defines the Host-to-Host NCHL online message specifications for both single message system and dual message system. Messages to be used for the connection between the NCHL host, issuer and acquirer will be based on the ISO-8583 standard. This document outlines the detailed usages of the ISO-8583 protocol between the two host systems and the data formats, field descriptions, codes, and files to be used in individual data elements. 

 ## 1.1	Audience

 This manual is intended for technical staff and managers and customer support personnel of NCHL member banks and member PSOs.

## 1.2 Organization of the Document


**Chapter 2, Introduction** – This chapter provides basic overview of the authorization system.

**Chapter 3, System Functionalities** – This chapter contains various functionalities of NCHL and members for handling online messages.

**Chapter 4, Message Structure** – This chapter contains the message structure supported by NCHL.

**Chapter 5, Message Format** – This chapter contains various message formats for various transactions.

**Chapter 6, Data Element Description** – This chapter defines the data element descriptions for NCHL NPS Card Switch Online messages.

**Chapter 7, Compliance** – This chapter contains the compliance requirements for NCHL members.

**Chapter 8, Responsibility** – This chapter contains the various responsibilities of NCHL members.
Annexure – This contains various supplement or appendix related with this document.
